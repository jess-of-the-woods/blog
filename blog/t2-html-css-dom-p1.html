A non-technical analogy of HTML, CSS & The Document Object Model (DOM)

I would say HTML is like the walls or structure of the house, the CSS is what determines the color, wallpaper, curtains etc. And the DOM is what it all looks like when complete, including dynamically generated elements like lighting, smoke effects etc.

Boxifying Design

Boxifying Design is where you look at a design mockup (pdf/jpg etc.) and then draw boxes around the different elements as a precursor and as an aid to creating it in html as an actual page.

The Box Model

The box model is a term used in the context of HTML/CSS in terms of layout. It is the idea that every element in a page is a box and each box has several items. These items are, from the outside in, the margin, the border, the padding and the content. The content is in the very middle surrounded by the other elements in concentric boxes.
